Basics Of Braided Messaging

  • A strand is a string of messages that are grouped together.
  • There can be many strands occurring in parallel, independent of one another.
  • By default, a message is not part of any strand (gray).
  • To create a new strand, prime a gray message (your friend's or your own) by clicking it, and then send a message while one or more messages are primed.
  • To view a strand, click any message in that strand.
  • To respond on a strand, send a message while viewing that strand.

Advanced Techniques

  • To remove a message from a strand, hold alt and click the message.
  • To add a message to a strand after the message has already been sent, prime the gray message, hold alt, and click any message in the desired strand.
